Admin-Bereich einrichten

Lege einen Zugangscode fest. Er wird verschlüsselt gespeichert — nicht im HTML.

Der Code wird als PBKDF2-Hash in deinem Browser gespeichert — nicht in der HTML-Datei. Dieses Gerät ist danach der Admin-Rechner.

Bitte Zugangscode eingeben um INI-Dateien zu verwalten.

Tutor-INI erstellen

Erstellt ein Schlüsselpaar für eine Lehrkraft. Die INI-Datei wird an Schülerinnen und Schüler weitergegeben. Die Daten können nur mit dem Masterpasswort entschlüsselt werden.

Masterpasswort sicher aufbewahren! Ohne es können Schüler-Daten nicht mehr entschlüsselt werden.
